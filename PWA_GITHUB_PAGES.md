# PWA через GitHub Pages

В папке `docs/` лежит статическая PWA-версия калькулятора. Ее можно опубликовать через GitHub Pages без сервера и без сборки.

1. Создай репозиторий на GitHub и загрузи проект.
2. Открой `Settings > Pages`.
3. В `Build and deployment` выбери `Deploy from a branch`.
4. В `Branch` выбери `main` и папку `/docs`.
5. Нажми `Save`.
6. Через несколько минут сайт будет доступен по адресу:
   `https://<твой-логин>.github.io/<имя-репозитория>/`

После открытия ссылки на Android можно добавить приложение на главный экран через меню Chrome.
